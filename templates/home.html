About the Project

This portal demonstrates a Flask-based machine learning workflow for pet disease prediction using symptom and profile data.

The goal is to provide an educational decision-support tool, not a replacement for professional diagnosis.

How the Model Works

  1. User submits pet information and symptoms.
  2. Input is transformed to model-ready features.
  3. Trained model predicts likely disease category.
  4. Result is shown as a supportive health insight.
